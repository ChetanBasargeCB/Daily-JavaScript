Welcome to Javascript Function World

  1. Function Parameter

    - A function can have multiple parameters.

    - Every parameter is a required parameter and has order dependency

  2. Rest Parameter

    - Every function can have only one “Rest” parameter.

    - Rest parameter must be defined along with other parameters, but it must be the

    last parameter in formal list.

  3. FAQ: What is difference between “Rest” & “Spread”?

    Ans: “Rest” is single parameter that can handle multiple arguments.

    “Spread” is one argument that can spread into multiple parameters.

  4. Function With Return

    5. A function is void by default.

    Void refers to “no-return”.

    - It will not return the memory of a function after the function call.

    - You can use “return” as jump statement inside function, so that it can return the function memory and alive in mempry.

  5. Function currying.

    A function returns another function within the context is known as Fuction currying

  6. Function Closure

    Closure is a mechanism of configuring function scope,

    Outer function member cand access inner function member that is known as Clouser

  7. Anonymous Function

    Anonymous Function

    - Anonymous functions a mostly used in function expression and function callback.

    - Anonymous function is enclosed in a “( )” and invoked by using IIFE technique.

    [Immediately Invoking Function Expression]

  8. Function Recursion:

    it is technique where function calls it self inside the context

    It is mostly used to create batch operations and design complex computations.

  9. Function Chaining

    - It is a mechanism of configuring a function that can invoke other functions in the context.

    - It will not have any order dependency in accessing functions a chain.

    - The context memory must be object type and can be accessed using “this” keyword.

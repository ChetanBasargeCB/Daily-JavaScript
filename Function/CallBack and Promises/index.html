Welcome to Function Callback & Promises Page

  1. Function Callback

    - Callback is technique of configuring a set of functions, so that it executes the matching

    - It uses “Synchronous” technique, which is a blocking mechanism.

    - It will block all other requests while processing the current request 

      Syntax=     function Name(f1, f2) {
                  f1(data);

                  f2(data)
          }

       Name(function(){ f1 data }, function(){ f2 data })

  2. Promises

    - Promise in JavaScript is just like a real world promise made to perform specified action.

    - It returns “resolve” on success and “reject” on failure.

    - It is an asynchronous function. It uses unblocking technique.

    - “Resolve” is configured using “then()” and “Reject” is configured using “catch()”.

    -“finally()” executes always.

    - You can configure multiple then() blocks to execute on success.

     Syntax =
              const name = new Promise(function(resolve, reject){

                   if(condition){

                       resolve(data);
                   }

                   else {

                       reject(error);

                   }
           })
           name.then(function(data){})

                   .catch(function(error){ })

                   .finally(function(){})

  3. Asynchronous Functions async & await

    - Function is by default synchronous and uses a blocking technique to access data.

    - You can make it “async” so that it can “await” while reading the data.

    - JavaScript ES6+ version makes all asynchronous functions as promise.

                    Syntax:

                   async function Name(){
                        return await value;
                                   }

  4. Pure and Impure Function

    -A pure function returns same result every time even when the parameter is changing.

    - An impure function returns different value every time event when the parameter is same.

                   Syntax Pure:
                function Demo(arg){
                    return "Hello World"
                }

                Syntax Impure: 
                function Demo(x){
                    let a=0
                    return a+=x
                }

  5. Arrow Functions

    - Arrow is a technique of writing function expression and callback in short hand.

                    Syntax Arrow:
             const add = (a, b) => a + b;

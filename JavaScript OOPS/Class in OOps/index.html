JavaScript OOPS

Class in OOPS

  • Class is a program template that provides data and logic.
  • It enables separation and code reusability.
  • Class is considered as an “Entity” when it is mapping to business requirements.
  • Class is considered as a “Model” when it is representing data requirements.

    • Class Members

    Class member can be any one of the following

    1. Property
    2. Accessor
    3. Method
    4. Consturctor

    1. Class Property

    • A property is responsible for handling data in class.
    • You can configure any type of data into property
    • You can access within or outside class.

      • Static in Property

    • JavaScript ES6+ version can make a property as static.
    • Static refers to continuous memory.
    • It is good for handling continuous operations.
    • A static member is accessible within or outside class by using class name.

      • Dynamic/ Non Static

    • All other members in class are by default “non-static” or dynamic.
    • Dynamic refers to “discreet memory”.
    • It is good for handling disconnected operations.
    • Non Static members are accessible within class by using “this” keyword and outside class by using instance of class. [object]

    2.Accessor

  • Accessors are used to have fine grained control over the property.
  • They are used for handling read and write operations on a property.
  • They are classified into 2 types
    1. get - getter
    2. Set - setter
  • “Getter” is used to read and return value of a property.
  • “Setter” is used to assign a new value into property.
